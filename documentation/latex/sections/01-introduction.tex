\subsection{Exercise Description}

This exercise demonstrates a secure user registration module with email verification, implementing modern web security practices. The registration module is responsible for:

\begin{itemize}[leftmargin=*,nosep]
  \item Secure user registration with client-side and server-side validation
  \item Password strength validation with detailed scoring (0--7 scale)
  \item Cryptographically secure verification token generation
  \item Email verification workflow before account activation
  \item Password hashing using Argon2 algorithm
  \item Defense against common attacks (enumeration, injection, brute force)
  \item Shared validation logic between frontend (WASM) and backend (native Rust)
\end{itemize}
